## Appendix A: Methodology

Each analysis loads data from the specified sources and computes:

1. **Archetype classification** — strategies classified by name pattern matching against known archetype families (momentum, mean_reversion, breakout, volatility, statistical_arb, multi_signal_ensemble, volatility_breakout)
2. **Indicator family classification** — indicator names extracted from strategy names/params and classified via substring matching
3. **WF pass rate** — fraction of strategies with `walk_forward_robust=True` in registry
4. **Overfitting signal** — Pearson correlation between discovery Sharpe (from winners.json) and WF test Sharpe (from sweep results or registry)
5. **Statistical significance** — insights require minimum 5 strategies and effect size >1.3x to be reported
